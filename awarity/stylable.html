Usage Rules

Blue familyDefault application theme. Use for UI chrome, interactive elements, and primary actions.
Amber familyAlternate warmth. Use for secondary branding contexts, alert-oriented dashboards, or user preference.
Dark teal surfacesShared across both families. The signature Awarity dark mode feel.
Charcoal #111827Icon fills, logo marks, and dark-on-light text. The anchor of the brand identity.

Brand: Theme Modes

Four application themes combine two color families (Blue, Amber) with two surface modes (Dark, Light). All themes share the same component library and token architecture — only the values change.

Blue Dark
Default theme. Deep teal surfaces, blue accents.
Blue Light
Clean white surfaces, blue accents for light environments.
Amber Dark
Same deep surfaces with warm amber accents.
Amber Light
Warm cream tones with amber accents for light environments.
